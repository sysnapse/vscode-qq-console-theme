# vscode-qq-console-theme
console theme (optimized) for vscode qq plugin | 可以查看圖片/發送圖片，查看视频，轉發消息，雙擊回復，發送表情包/emoji 的 console 主題


### 内置功能

(无括号则与原版使用方式相同)

- 使用 Enter 发送 而不是 ctrl + Enter

- 查看图片

- 发送黏贴图片

- 查看视频(点击展开, 无需下载)

- 查看转发消息

- 双击聊天信息进行艾特回复

- 双击群友名字进行艾特

- 发送表情包/emoji

- 查看app分享
